🍲 Recipe Finder Web App

A modern and responsive Recipe Finder Web Application built using HTML, CSS, and JavaScript.
Users can explore different recipes, search for dishes, and view detailed ingredients in a clean modal interface.

🌐 Live Demo:
👉 https://recipe-alpha-seven.vercel.app

🚀 Features

🔍 Search recipes instantly

📂 Category-based filtering

🖼️ Interactive recipe cards

📋 Detailed ingredients modal

📱 Fully responsive design

⚡ Fast and lightweight (No frameworks used)

🛠️ Tech Stack

Frontend: HTML5, CSS3, JavaScript (Vanilla JS)

Deployment: Vercel

Version Control: Git & GitHub
